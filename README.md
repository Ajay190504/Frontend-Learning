# Frontend Development Learning Repository

## Overview

This repository documents my frontend development learning journey through structured notes, coding exercises, assignments, practice questions, and mini projects.

The repository covers the fundamental technologies required for modern web development, including HTML, CSS, JavaScript, React, and SQL. Each section contains learning materials, hands-on practice tasks, and examples created while building a strong foundation in frontend engineering.

This repository serves as both a personal knowledge base and a reference resource for future projects, interview preparation, and continuous learning.
